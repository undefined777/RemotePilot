# RemotePilot

一个美观的 Windows 远程控制工具。

## 技术栈

- **桌面端**: Electron + React + TypeScript
- **后端**: Node.js + Express + WebSocket
- **iOS**: 快捷指令 (Shortcuts)

## 模块说明

```
RemotePilot/
├── backend/          # 后端服务 (Node.js)
│   └── web-ui/       # Web 管理界面
├── desktop/          # Windows 桌面客户端 (Electron)
└── ios/              # iOS 快捷指令
```
